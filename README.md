# ATM Interface

## Introduction

This is a simple ATM Interface project developed using Java. The project is console based and allows a user to perform basic ATM operations after logging in with a valid User ID and PIN.

The purpose of this project is to understand Java concepts such as classes, objects, methods, arrays/lists, and menu-driven programming.

## Features

- User Login using User ID and PIN
- View Transaction History
- Withdraw Money
- Deposit Money
- Transfer Money
- Exit the Application

## Project Structure

```
src
│
├── model
│   └── Account.java
│
├── service
│   └── ATMService.java
│
└── Main.java
```

## Default Login Details

User ID: admin

PIN: 1234

## How to Run

1. Open the project in VS Code or any Java IDE.
2. Compile all Java files.
3. Run the Main.java file.
4. Enter the User ID and PIN.
5. Select the desired operation from the menu.

## Concepts Used

- Java Programming
- Object Oriented Programming
- Classes and Objects
- ArrayList
- Conditional Statements
- Loops

## Output

After successful login, the user can perform ATM operations such as deposit, withdrawal, transfer, and view transaction history through a menu-driven interface.
